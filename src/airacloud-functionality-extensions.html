Creating the extension

For creating a functionality airacloud extension, you can:

On Windows / Mac

Create a folder with the name of your plugin and inside the folder create this files:
  • manifest.json: A file with information about your extension
  • main.js: The main javascript file
  • README.md: A README File that shows up in the page of your extension.

Inside manifest.json set this code:

{
  "name": "My extension",
  "main": "main.js",
  "author": "username",
  "desc": "Example plugin for AiraCloud"
}
